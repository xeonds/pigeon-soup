# 线性结构

就是数据线性排列的结构。一般有这几种：

## 数组

和C中的一样，下标引用数组中的任意一项。但是向其中某一位置插入数据时，需要将其后边所有的数后移一个。

### 链表

类似数组，不过没有直接引用，元素之间靠指针来连接。每个元素都有指向自己前驱/后继的指针，它插入数据不需要移动后面的元素，只需要改变插入位置前后元素的指针即可。但是它的查找必须从一个元素开始，一个一个查找。

## 栈
